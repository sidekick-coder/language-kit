MarkdownNodeHeading

Reference

Node use to represent headings.

Properties

toHtml

  • Type: function

Convert the node to Html text

node.toHtml() // <h1>Heading</h1>

level

  • Type: number

Level of the heading, Example:

# Heading 1 > level: 1
## Heading 2 > level: 2
### Heading 3 > level: 3
